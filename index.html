Developer Reference
Security Architecture
Backend security blueprint · Secret management · Compliance requirements
Security Measures
🔑
API Key Management
All secrets in environment variables — never hardcoded. Use dotenv locally, AWS SSM / HashiCorp Vault in production.
🔒
Database Encryption (AES-256)
KYC documents and PII encrypted at rest. Sensitive data in an isolated, access-controlled database with field-level encryption.
🛡
JWT + 2FA Authentication
Short-lived access tokens (15 min) + rotating refresh tokens. TOTP-based OTP required for every buy/sell transaction.
🌐
HTTPS / TLS 1.3 Only
All traffic over TLS 1.3. HSTS max-age 1 year, strict CSP headers, X-Frame-Options DENY.
📋
Immutable Audit Log
IP address, timestamp, user ID, and action type logged for every event. Append-only storage — records cannot be modified retroactively.
🚦
Rate Limiting & DDoS Protection
100 req/min per IP on API routes. Cloudflare WAF in front of all endpoints. Suspicious activity triggers automatic account lock.
